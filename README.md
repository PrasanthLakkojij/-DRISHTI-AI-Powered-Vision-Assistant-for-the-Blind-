# 👁️ DRISHTI - AI Visual Assistant

## Overview

DRISHTI is an AI-powered visual assistant developed to assist visually impaired individuals in understanding and navigating their surroundings.

The system combines computer vision, artificial intelligence, speech technologies, and real-time environmental analysis to provide spoken guidance and scene awareness.

---

## Features

### 🔍 Object Detection
- Detects people, vehicles, obstacles, furniture, and daily-use objects.
- Real-time identification using YOLO.

### 📏 Distance Estimation
- Estimates distance between user and detected objects.
- Warns about nearby obstacles.

### 👤 Face Recognition
- Identifies known persons.
- Alerts when recognized individuals appear.

### 📖 OCR Text Reading
- Reads printed text from books, labels, signs, and documents.
- Converts text into speech.

### 🎙️ Voice Interaction
- Accepts voice commands.
- Provides spoken feedback.

### 🧭 Navigation Assistance
- Guides users through indoor and outdoor environments.
- Detects obstacles and safe directions.

### 🌍 Scene Understanding
- Describes surroundings in natural language.

### 🔊 Telugu Voice Support
- Supports Telugu speech output.
